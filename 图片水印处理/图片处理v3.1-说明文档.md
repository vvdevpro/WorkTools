# 🖼️ 图片处理全能助手 v3.1 — 使用说明文档

## 📋 概述

**图片处理全能助手 v3.1** 是一个基于 Python + Tkinter 的桌面 GUI 工具，提供三大核心图片处理功能：

1. **📍 添加时间/地点水印** — 在图片底部居中位置叠加时间、日期、地点信息
2. **🛡️ 添加满屏防盗水印** — 生成 45° 倾斜平铺的满屏防盗文字水印
3. **🗜️ 批量无损图片压缩** — 二分查找算法自动匹配最优压缩质量，将图片压缩到指定 KB 以内

支持格式：`.jpg` / `.jpeg` / `.png` / `.bmp` / `.webp`

---

## 🚀 快速开始

### 1️⃣ 环境要求

- Python 3.8+
- 依赖库：`Pillow`

安装依赖：

```bash
pip install Pillow
```

### 2️⃣ 运行方式

将 `图片处理v3.1.py` 与要处理的图片放在 **同一文件夹** 下，直接运行：

```bash
python 图片处理v3.1.py
```

或双击 `.py` 文件（需已关联 Python 解释器）。

### 3️⃣ 程序界面

启动后出现主窗口，包含三个功能按钮和一个提示信息：

## 📍 功能一：时间/地点水印

### 配置界面

| 参数 | 说明 | 示例 |
|------|------|------|
| **初始时间** | 水印显示的时间（24小时制，可选留空） | `09:51` |
| **日期** | 水印显示的日期文字（可选留空） | `2026-04-16 星期四` |
| **地点** | 水印显示的地点文字（可选留空） | `重庆市·gay佬转转转酒吧` |
| **时间动态随机增加** | 勾选后每张图片的时间随机增加 1~4 分钟 | ✅ |

### 水印布局效果

```
┌──────────────────────────────┐
│                              │
│           (图片内容)          │
│                              │
│                              │
│           09:51              │  ← 大号字体，底部约 75% 位置
│                              │
│   2026-04-16 ● 重庆市...     │  ← 小号字体 + 红色圆点分隔
└──────────────────────────────┘
```

### 处理逻辑

1. 遍历当前目录下所有图片文件
2. 自动校正 EXIF 方向信息
3. 根据图片 `min(宽, 高)` 动态计算字体大小
   - 大号字体：`base_size × 0.15`（用于时间）
   - 小号字体：`base_size × 0.038`（用于日期/地点）
4. 若勾选了"时间动态随机增加"，每处理一张图片，时间递增 `random(1, 4)` 分钟
5. 输出到 `已加时间地点水印/` 目录

### 字体渲染

- 优先使用微软雅黑 (`msyh.ttc`)，回退到黑体 (`simhei.ttf`)
- 白色文字 + 1px 黑色描边，提升在各种背景下的可读性

## 🛡️ 功能二：满屏防盗水印

### 配置界面

| 参数 | 说明 | 默认值 |
|------|------|--------|
| **防盗水印文字** | 平铺显示的文字内容 | `内部文件 请勿外传` |
| **不透明度** | 滑块调节，范围 10~255 | `100` |

### 水印布局效果

```
  ╲           ╲
   ╲ 防盗文字 ╲ 防盗文字         ← 45° 倾斜平铺
    ╲         ╲
 防盗文字  防╲盗文字  防盗文字
            ╲
  防盗文字  ╲防盗文字 防盗文字
            ╲
```

### 处理逻辑

1. 创建一张足够大的透明图层（边长 = 图片对角线长度）
2. 在小号字体模式下，交错平铺水印文字（每行错开半个间距）
3. 避开中心区域，在中心放置大号水印文字
4. 将水印图层旋转 45° 后与原图 `alpha_composite` 叠加
5. 输出到 `已加防盗水印/` 目录

### 技术细节

- 中心大号字体尺寸：`base_size × 0.05`
- 周边小号字体尺寸：`base_size × 0.04`
- 间距：`字宽 + base_size × 0.35`
- 平铺行间交错偏移：`间距 / 2`
- 中心区域保护半径：大字宽度 × 0.5 + base_size × 0.1

## 🗜️ 功能三：批量无损图片压缩

### 配置界面

| 参数 | 说明 | 默认值 |
|------|------|--------|
| **限制最大体积 (KB)** | 目标文件大小上限 | `100` |

### 压缩策略（二分查找算法）

```
输入: 目标 ≤ 100 KB
  ↓
1. 先用 quality=95 尝试
   → 如果 ≤ 100 KB，直接返回
   ↓ 否则
2. 初始化 low=10, high=95
3. while low ≤ high:
     mid = (low + high) // 2
     用 quality=mid 压缩并检查大小
     如果 ≤ 目标 → low = mid + 1 (提高质量)
     如果 > 目标  → high = mid - 1 (降低质量)
   ↓
4. 使用 best_quality 输出最终文件
```

### 预处理

- 若图片最大边长 > 1920px，先等比缩放到 1920px
- RGBA/P 模式自动转换为 RGB
- 输出为 JPEG 格式，`subsampling=0`（保留色彩细节）
- 输出到 `已压缩结果/` 目录
- 输出文件名格式：`原文件名_已压缩.jpg`

- 配置在每次点击"保存并开始处理"时自动更新
- 下次启动程序自动加载上次保存的配置

## ⚠️ 注意事项

1. **图片位置**：程序只处理与 `.py` 文件 **同一目录** 下的图片
2. **字体依赖**：依赖 Windows 系统字体，在非 Windows 系统上需修改 `FONT_PATH`
3. **EXIF 自动校正**：使用 `ImageOps.exif_transpose()` 自动翻转方向
4. **压缩格式**：压缩功能输出均为 JPEG 格式，原 PNG 透明通道会被舍弃
5. **输出目录自动创建**：各功能输出目录在首次运行时自动创建
